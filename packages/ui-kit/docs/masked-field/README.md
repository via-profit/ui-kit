## Описание

Утилиты для создания маскированных текстовых полей.  
Состоят из двух частей:

- **`useMasked`** — хук, который парсит и форматирует ввод по маске.
- **`MaskedField`** — готовый компонент ввода, который использует хук и работает как обычный `<input>`, но с маской.

Подходит для телефонов, дат, банковских карт, паспортов, кастомных форматов и любых масок, основанных на `RegExp`.

---

## Возможности

- Маскирование ввода по массиву `Mask`
- Поддержка фиксированных символов (`'+'`, `'('`, `')'`, `'-'` и т. д.)
- Поддержка `RegExp` для динамических символов
- Автоматическое форматирование текста
- Корректная работа с caret (позиция курсора)
- Поддержка динамических масок через функцию `GetMask`
- Возможность подменить парсер (`parseInput`)
- Возможность трансформировать итоговое значение (`transform`)
- Полная контролируемость (как у обычного `<input>`)

---

## Mask

Маска — это массив, состоящий из:

- **строковых паттернов** — фиксированные символы, которые автоматически вставляются
- **RegExp паттернов** — символы, которые должен ввести пользователь

Пример маски телефона:

```ts
const phoneMask: Mask = [
  '+',
  '7',
  ' ',
  '(',
  /\d/,
  /\d/,
  /\d/,
  ')',
  ' ',
  /\d/,
  /\d/,
  /\d/,
  '-',
  /\d/,
  /\d/,
  '-',
  /\d/,
  /\d/,
];
```

# useMasked

Низкоуровневый хук, который предоставляет три функции:

```ts
const { parseInput, formatParsedInput, parseAndFormat } = useMasked();
```

### `parseInput(value, mask, caret)`
Парсит ввод, удаляет лишние символы, оставляет только те, что подходят под маску.

### `formatParsedInput(parsedValue, mask, caret)`
Форматирует строку согласно маске, добавляет фиксированные символы.

### `parseAndFormat(value, mask, caret)`
Комбинирует оба шага.

# MaskedField

Готовый компонент, который использует `useMasked` внутри.

## Пример использования

```tsx
import { MaskedField } from 'your-lib';

const phoneMask = [
  '+',
  '7',
  ' ',
  '(',
  /\d/,
  /\d/,
  /\d/,
  ')',
  ' ',
  /\d/,
  /\d/,
  /\d/,
  '-',
  /\d/,
  /\d/,
  '-',
  /\d/,
  /\d/,
];

export default function App() {
  const [value, setValue] = React.useState('');

  return (
    <MaskedField
      label="Телефон"
      value={value}
      mask={phoneMask}
      onChange={({ text }) => setValue(text)}
    />
  );
}
```

# Props

### `value: string | null`
Текущее значение поля.

### `mask: Mask | GetMask`
Маска или функция, возвращающая маску.

Пример динамической маски:

```ts
const getMask: GetMask = (input) => {
  if (input.startsWith('8')) {
    return ['8', ' ', /\d/, /\d/, /\d/, '-', /\d/, /\d/, '-', /\d/, /\d/];
  }
  return phoneMask;
};
```

### `onChange(payload)`
Вызывается при каждом изменении ввода.

`payload` содержит:

```ts
{
  text: string;   // отформатированное значение
  caret: number;  // позиция курсора
  isValid: boolean; // соответствует ли маске
}
```

### `parseInput?: ParseInput`
Можно переопределить парсер.

### `transform?: (value: string) => string`
Позволяет изменить итоговое значение перед отдачей наружу.

Пример: убрать пробелы

```ts
transform={(v) => v.replace(/\s+/g, '')}
```

# Динамические маски

Можно менять маску в зависимости от ввода:

```ts
const getMask: GetMask = (input) => {
  if (input.startsWith('+1')) return usMask;
  return ruMask;
};
```

---

# Трансформация значения

Можно изменить итоговое значение перед отдачей наружу:

```ts
<MaskedField
  transform={(v) => v.replace(/\D/g, '')} // оставить только цифры
/>
```

---

# Кастомный парсер

Если стандартный `parseInput` не подходит:

```ts
<MaskedField
  parseInput={(value, mask, caret) => {
    // кастомная логика
    return { text: value, caret };
  }}
/>
```

---

# Ограничения и поведение caret

- caret автоматически перескакивает через фиксированные символы маски
- при удалении фиксированные символы не удаляются
- маска не раскрывается, пока пользователь не ввёл первый символ
